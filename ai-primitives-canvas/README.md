# AI Integration Canvas

A brainstorming tool that helps managers explore how AI can transform their work.

## Features

- **Smart Onboarding**: 3 quick questions to understand your role and goals
- **AI-Generated Ideas**: Instantly generates 2-3 tailored ideas per category
- **6 AI Integration Categories**:
  - Content Creation
  - Task Automation
  - Research & Synthesis
  - Data & Insights
  - Technical Work
  - Strategy & Ideation
- **Persistent Triage**: Star, edit, or dismiss any idea at any time
- **AI Chat**: Go deeper in any category with contextual brainstorming
- **Export**: Download your action plan as Markdown

## Getting Started

### Prerequisites

- Node.js 18+
- An Anthropic API key

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```
